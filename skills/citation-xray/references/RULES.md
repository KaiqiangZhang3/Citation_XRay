# Classification and Counting Rules

## Counting rules

### Citation occurrence

Count each independent source occurrence at each relevant citation location, after splitting footnotes and resolving cross-references.

### Unique source

Count a source once across the chosen scope, regardless of repeated appearances.

### Cross-reference rule

Do not count internal references as sources. Resolve them first.

Examples not counted as sources by themselves: see MN 105; see n. 143; ibid.; supra fn. 168; see Art. 53; infra MN 16.

## Countability by citation function

| Citation function | Count by default? |
|---|---|
| core_support | yes |
| auxiliary_support | yes |
| historical_background | no |
| contrary_or_reserved_view | no |
| neutral_context | no |
| internal_cross_reference | no |
| duplicate_already_counted | no |
| uncertain | no |

## Signal phrases indicating caution

If a source is introduced by any of these, do not classify it as support without checking context: but see; however; on the other hand; critically; unconvincing position; contrary view; different view; disputed; retained significance; still applicable; not affected.

## Source type rules

### scholarly_writing

Includes monographs, law review articles, book chapters, treatises, article-by-article commentaries, encyclopedia entries, scholar-edited practice reviews if cited as academic synthesis.

Examples: Kelsen; Blumenwitz; Hacker; CP/Kodjo; CPF/Kolb; Zündorf; Tomuschat; Verdross/Simma; Brownlie; M Wood.

### treaty_or_formal_instrument

Includes treaty text, convention text, formal agreement, UNTS/ILM treaty document, official constitutional or charter text.

Does not include a book about a treaty, commentary on a treaty, or scholar discussing a formal instrument.

### government_or_state_practice

Includes aide-mémoire, government declaration, state representative speech, official state memorandum, foreign office document, national position paper.

A state practice source can still be non-countable if it is contrary or reserved.

### international_organization_document

Includes UNGA resolutions, UNSC resolutions, UN reports, official UN documents, official records of UN meetings, reports of UN committees.

### case_or_judicial_material

Includes ICJ, PCIJ, ITLOS, PCA, arbitration awards, domestic judgments if relevant, advisory opinions.

## Origin rules

The user must choose strict nationality/institutional origin, legal tradition, or project-specific origin categories. If not specified, use a practical taxonomy and note uncertainty.

## Mixed sources

If one row contains multiple independent sources, split it. Do not classify the row as one blended source.

## Bibliography rule

A source listed only in a bibliography is not counted unless cited in a relevant footnote.
